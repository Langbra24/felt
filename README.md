# felt

A public lab exploring AI epistemic state through generative visuals.

Built with p5.js, Tweakpane, and the Anthropic streaming API.

## experiments
- `01-boids` — Boids flocking simulation, foundation for AI-driven parameter control

## run locally
```
npm install
npm run dev
```

## stack
- p5.js (via CDN)
- Tweakpane (via CDN)
- Vite (dev server + build)
- GitHub Pages (hosting)
